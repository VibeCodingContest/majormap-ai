# MajorMap AI

복수전공·심화전공 대학생을 위한 AI 진로-교과 로드맵 설계 서비스

MajorMap AI는 대학생의 학번, 전공 조합, 수강 이력, 관심 분야를 바탕으로 현재 적합한 진로를 추천하고, 부족한 역량과 다음 학기에 들으면 좋은 과목까지 연결해서 보여주는 AI 기반 웹 서비스입니다.

## 1. 프로젝트 소개

대학생은 과목을 많이 들어도 그것이 어떤 진로와 연결되는지 파악하기 어렵습니다. MajorMap AI는 수강 이력을 진로 언어로 다시 해석해, 학생이 이미 쌓은 역량, 가능한 진로 후보, 부족한 역량, 다음 학기 추천 과목까지 한 흐름으로 보여줍니다.

## 2. 실행 방법

```bash
npm install
cp .env.example .env.local
npm run dev
```

로컬에서는 `http://localhost:3000`에 접속하면 됩니다.

배포 데모: [https://majormap-ai.vercel.app/](https://majormap-ai.vercel.app/)

## 3. 문제 정의

- 복수전공·심화전공 학생은 자신이 들은 과목이 어떤 진로와 연결되는지 파악하기 어렵습니다.
- 학번별 커리큘럼 차이 때문에 선배의 조언이나 일반적인 과목 추천이 그대로 적용되지 않는 경우가 많습니다.
- 대학 시스템은 졸업요건 확인에는 도움을 주지만, 교과 이력을 진로 관점에서 해석해주지는 못합니다.

## 4. 해결 아이디어

사용자가 학번, 주전공, 복수전공 또는 심화전공, 수강 과목, 관심 키워드를 입력하면 시스템이 구조화된 과목-역량-진로 데이터를 바탕으로 적합도를 계산합니다. 이후 각 진로에 대해 추천 이유, 보유 역량, 부족 역량, 미수강 과목, 다음 학기 추천 과목까지 연결해서 보여줍니다.

## 5. 주요 기능

- 학번, 전공 조합, 수강 과목, 성적, 관심 키워드 입력
- Top 3 진로 추천
- 적합도, 추천 확신도, 보유 역량, 부족 역량 표시
- 핵심 미이수 과목 및 보완 과목 제안
- 다음 1~2개 학기 수강 계획 생성
- AI 기반 추천 이유 및 로드맵 해설

## 6. 대상 사용자

- 복수전공 또는 심화전공 중인 대학생
- 학번별 커리큘럼 차이 때문에 과목 선택이 어려운 학생
- 현재 수강 이력을 진로와 연결해 보고 싶은 학생
- 대학 진로지도, 튜터링, 상담에 활용할 교육 담당자

## 7. 서비스 흐름

1. `/recommend`에서 학생 정보와 수강 이력을 입력합니다.
2. `POST /api/recommend`가 규칙 기반으로 진로 적합도를 계산합니다.
3. 결과 화면에서 추천 진로, 보유 역량, 부족 역량, 추천 과목을 확인합니다.
4. 필요하면 AI 해설로 추천 이유를 자연어로 확인합니다.
5. 선택한 진로를 기준으로 `POST /api/plan`이 다음 1~2개 학기 계획을 생성합니다.

### 예시

입력:

```text
학번: 2024
주전공: 컴퓨터공학
복수전공: 경영학
수강 과목: 프로그래밍입문, 데이터구조, 경영학원론, 마케팅, 데이터분석개론
관심 키워드: 서비스기획, 데이터
```

출력:

```text
추천 진로: 프로덕트 매니저
보유 역량: 프로그래밍, 비즈니스 이해, 데이터 분석
부족 역량: 프로젝트 관리, 전략 심화
추천 과목: IT프로젝트관리, 경영전략
```

## 10. 추천 시스템 동작 방식

추천 시스템은 과목, 역량, 진로를 구조화한 데이터에 기반해 동작합니다.

- 과목마다 역량 태그를 부여합니다.
- 진로마다 필수 역량과 선택 역량을 정의합니다.
- 학생이 이수한 과목에서 보유 역량을 추출합니다.
- 보유 역량과 진로 요구 역량을 비교해 적합도를 계산합니다.

현재 점수 계산에는 필수 역량 커버리지, 선택 역량 커버리지, 전공 적합도, 관심 키워드 보너스, low-evidence penalty, 핵심 과목 성적 반영이 포함됩니다.

LLM은 추천 이유 설명, 부족 역량 해설, 로드맵 문장화에만 사용되며, 추천 점수 계산 자체는 규칙 기반으로 처리됩니다.

## 11. MVP 범위와 한계

### 현재 MVP 범위

- 특정 대학 1곳을 가정한 샘플 커리큘럼 1종
- 학과군 2개: 컴퓨터공학, 경영학
- 학번 트랙 2개: 2023, 2024
- 진로 4개
- 추천 결과와 다음 1~2개 학기 계획

### 현재 제외한 범위

- 실제 학사 시스템 연동
- 로그인 및 회원가입
- 전체 대학 및 전체 학과 확장
- 실제 채용공고 크롤링


## 12. 향후 개선 방향

- 실제 대학 교과과정 데이터 연동
- 학과군 및 진로 수 확장
- 졸업요건, 선수과목, 학사 규정의 정교한 반영
- 비교과 활동, 프로젝트 경험, 자격증 데이터 연결
- 진로센터 및 지도교수 상담 보조 도구로 확장
