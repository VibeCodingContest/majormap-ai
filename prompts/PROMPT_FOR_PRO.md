# ChatGPT Pro용 프롬프트

너는 지금 **KIT 바이브코딩 공모전** 팀의 기술 PM + AI 협업 코치 역할을 맡는다.

## 전제
- 팀원은 2명이다.
- 나의 개발환경: **Cursor + Cursor Agent + Cursor의 Codex 연동 가능 + ChatGPT Pro 사용 가능**
- 친구의 개발환경: **VS Code + Codex 사용**
- GitHub는 **Cursor의 Codex에 연결되어 있음**
- 공모전은 **AI 활용 차세대 교육 솔루션** 주제이며, 단순 LMS가 아니라 교육 현장의 페인 포인트를 해결하는 서비스가 중요하다.
- 이번 프로젝트 아이디어는 아래와 같다.

## 프로젝트 아이디어
**복수전공/심화전공 대학생을 위한 AI 진로-교과 로드맵 설계 서비스**

핵심 문제:
- 대학생은 자신이 들은 과목이 어떤 세부 진로와 연결되는지 잘 모른다.
- 복수전공/심화전공 학생은 특히 과목 조합이 진로에 어떻게 이어지는지 파악하기 어렵다.
- 학번마다 교육과정(커리큘럼)이 달라, 단순 과목 추천은 실제 상황에 맞지 않을 수 있다.

핵심 해결:
- 사용자가 **학번, 주전공, 복수전공/심화전공, 이미 들은 과목, 관심 진로 키워드**를 입력하면
- AI가 **가능한 진로/세부분야를 추천**하고
- 각 진로별로 **왜 적합한지 근거를 설명**하고
- **부족한 역량**과 **미수강 추천 과목**을 제안하는 서비스

## 중요한 방향
- 대상은 우선 **대학교** 중심으로 한다.
- MVP는 **특정 대학/특정 학과군/특정 학번**으로 제한해서 완성도를 높인다.
- LLM이 모든 판단을 다 하는 구조가 아니라,
  - **학번별 커리큘럼 / 과목 / 역량 / 진로 연결은 구조화 데이터 + 규칙 기반**
  - **설명 문구 생성, 추천 근거 해설, 로드맵 문장화는 LLM 기반**
  으로 구성한다.
- 배포는 웹사이트 형태로 해야 하므로 **Next.js + Vercel** 중심으로 생각한다.
- 공개 GitHub 저장소 제출 예정이므로 **API 키 노출 방지**가 중요하다.
- 공모전 특성상 **AI와 협업한 흔적이 보이는 문서/파일을 저장소에 포함**하는 전략도 필요하다.
