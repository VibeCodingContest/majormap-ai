# FEEDBACK REQUEST

친구에게 보낼 짧은 테스트 요청문과 확인 항목을 정리한 문서다.

---

## 친구에게 보낼 짧은 문구

> 잠깐만 `trycloudflare` 임시 링크로 테스트해줘.  
> 첫 화면, 진로 추천, 다음 학기 계획, AI 해설이 정상 동작하는지만 확인해주면 돼.  
> 이상하면 어떤 버튼을 눌렀는지랑 화면 캡처만 보내줘.

---

## 확인 부탁할 항목 5개

1. 첫 화면이 자연스럽게 열리는지
2. `/recommend`로 들어가서 추천이 나오는지
3. `진로 추천 받기`를 눌렀을 때 결과 카드가 보이는지
4. `이 진로로 계획 짜기`와 `다음 학기 계획 생성`이 동작하는지
5. `AI 해설 보기`를 눌렀을 때 에러 없이 열리거나, 최소한 카드 안에서만 안전하게 실패하는지

---

## 에러가 나면 보내달라고 할 것

- 어떤 버튼을 눌렀는지
- 에러 화면 전체 스크린샷
- 브라우저 주소창 URL
- 가능하면 개발자도구 Network 탭 캡처
- 새로고침 직후인지, 이전 탭을 재사용했는지도 함께 알려달라고 요청

---

## 같이 안내하면 좋은 말

- 가능하면 시크릿 창에서 열어달라고 한다.
- 예전 링크가 있으면 그건 닫고 새 링크만 써달라고 한다.
- 모바일로 볼 경우 화면이 좁아질 수 있으니, 에러가 나면 데스크톱에서도 한 번 확인해달라고 한다.
