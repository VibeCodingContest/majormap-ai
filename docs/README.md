# DOCS INDEX

이 폴더는 `majormap-ai` canonical repo 기준 문서만 유지한다. 과거 초안 폴더인 `초기설계안`은 source of truth가 아니며, 필요한 문서만 정리해 이 위치로 흡수한다.

---

## 제품/범위

- `PLAN.md`: 문제 정의와 해결 방향
- `PROJECT_SCOPE.md`: MVP 포함/제외 범위
- `STATUS_HANDOFF.md`: 현재 저장소 상태 요약
- `TASKS.md`: 남은 작업과 완료 상태

## 운영/협업

- `GIT_WORKFLOW.md`: 브랜치 전략과 제출 전 동결 절차
- `AI_COLLAB.md`: AI 협업 기록
- `PROMPT_LOG_TEMPLATE.md`: 프롬프트 로그 기록 양식

## 제출/배포

- `FINAL_CHECKLIST.md`: 제출 직전 체크리스트
- `SUBMISSION_GUIDE.md`: public repo, live URL, 제출물 정리 가이드
- `PROD_QA.md`: live URL 기준 최종 수동 점검 문서
- `DEMO_BLOCKERS.md`: 현재 데모 리스크와 확인 포인트

## 심사용 문서

- `AI_REPORT_DRAFT.md`: AI report PDF 원문 초안
- `JUDGE_ONE_PAGER.md`: 심사용 1페이지 요약

## 참고

- `DEMO_SCRIPT.md`: 추천 흐름 기준 클릭 순서 참고 문서
