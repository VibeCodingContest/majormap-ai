# PLAN

## 문제 정의
복수전공/심화전공 대학생은 수강 이력과 진로의 연결을 파악하기 어렵다.
특히 학번별 커리큘럼 차이 때문에 선배의 과목 추천이 그대로 통하지 않는 경우가 많다.

## 해결 방향
- 구조화된 커리큘럼/과목/역량/진로 데이터 준비
- 규칙 기반 적합도 계산
- LLM으로 추천 이유와 학습 로드맵 설명 생성

## 핵심 사용자
- 3~4학년 대학생
- 복수전공/심화전공 학생
- 진로 미정이지만 관심 분야가 있는 학생

## 핵심 사용자 가치
- 내가 이미 쌓은 역량을 한눈에 파악
- 가능한 진로 후보 확인
- 부족한 역량을 채우는 과목 추천
